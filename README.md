## Objectives

The porpouse of challenge is consolidate knowledge in creating a continous integration pipeline by following.
1. Create a simple node.js application
2. Create unit tests for this application
3. Create a CI pipeline using Github actions that:
 3.1 Install the application
 3.2 Run the tests
 3.3 Run SonarCloud
4. Upload this application to a public repository on Github and activate the status check so that it can only be merged if the application passes the CI pipeline and also if the application passes SonarCloud's quality gate.
5. Create a Pull Request (PR) carrying out the whole process.
